My Processing Experience

Processing is a language based on Java that is specifically geared towards creating graphical code. Many people think languages like Python, Java, or even Scratch should be a person's first language. I believe it should be Processing. Using processing lets you learn all the knitty gritty aspects of coding in a real language (opposed to block code) but also lets you see what you make in terms of objects on a screen instead of print texts

I learned Processing when I was 11 and it is by far still my favorite language to play around with. Over the years I have done a few animations in Processing and even used it to make live wallpapers for my phone.
